WHAT IS SPEAKUP AI?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SpeakUP AI is a personalized coaching platform for high school and college
students learning debate, Model UN, and public speaking. Students complete a
short onboarding survey and receive a custom learning plan built around their
goals and experience level. The AI is hidden from the user — it presents
itself as a personal coaching platform, not an AI product.

AI FEATURES DETAIL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DEBATE FEATURES (inside DebateArena tabs):

Sparring Partner
  - Student picks motion and side (AFF/NEG)
  - AI plays the opposing side in real-time chat
  - Fires POIs, attacks weak points, stays in character
  - Level 1: friendly / Level 2: strategic / Level 3: relentless
  - After 3+ exchanges: gavel icon triggers AI Judge Ballot
  - Judge Ballot: winner, score/100, reasoning, clash analysis,
    impact weighing, main improvement
  - Session recorded to progress tracker

Rebuttal Trainer
  - Random argument appears (level-appropriate topics)
  - 90-second countdown timer
  - 3 rebuttal text inputs
  - AI grades each rebuttal individually + gives improved version
  - Returns: overall score, per-rebuttal score + feedback + rewrite,
    best rebuttal, biggest weakness, next drill
  - Session recorded

Case Builder
  - Student selects motion + side (Proposition/Opposition)
  - Level-appropriate motion suggestions
  - AI builds: framing, mechanism, 2-3 arguments (with examples + impacts),
    weighing statement, full opening speech script
  - Session recorded

Speech Feedback (6 areas)
  - Student writes speech for a topic
  - Speech type selector (Constructive / Rebuttal / Summary / Final Focus)
  - AI scores each of 6 dimensions: Structure / Argument / Evidence /
    Rebuttal / Impact / Delivery
  - Each dimension: score/10 + specific feedback + one tip
  - Plus: improved paragraph rewrite
  - Session recorded

Weakness Detector
  - Reads all session data from localStorage
  - Surfaces top 3 weaknesses by frequency
  - Each weakness gets a targeted drill
  - Shows recent sessions with scores
  - Updates after every session

Progression System
  - XP earned per session: sparring=25, speech=18, rebuttal=12, case=10
  - Progress bar fills to 100%
  - At 100%: Gemini evaluates last 5 sessions + top weaknesses
  - Returns: advance/stay/drop verdict + reasoning + strengths + gaps + next focus
  - Advance: moves to next level, bar resets
  - Stay: bar resets at same level
  - Drop: rare — returns to Level 1
  - Now knows full curriculum → references specific lessons in feedback

SPEAKING FEATURES (inside SpeakingArena tabs):

Speech Upload (Speech Feedback tab)
  - Student writes speech for a level-appropriate topic
  - AI scores: Articulation / Structure / Style / Delivery / Persuasion / Confidence
  - Each: score/10 + specific feedback + one improvement tip
  - Plus: improved opening rewrite
  - Session recorded

Event Practice (10 events)
  - Student picks event type from 10 options
  - Tips expand for each event format
  - AI evaluates at event-specific standard:
    Self-Introduction: clarity, warmth, memorability (60 sec)
    Class Presentation: structure, evidence, engagement (3-5 min)
    Valedictorian Speech: story, inspiration, personal voice (3-4 min)
    TEDx-Style Talk: big idea, narrative arc, delivery energy (5-7 min)
    Persuasive Speech: ethos/pathos/logos + call to action (3-5 min)
    Informative Speech: clarity, organization, audience (3-5 min)
    Debate Opening Statement: framing, confidence, preview (2-3 min)
    Model UN Opening Speech: formal language, representation (1-2 min)
    College Interview Answer: specificity, authenticity (60-90 sec)
    Elevator Pitch: hook, value prop, close (30-60 sec)
  - Returns: judge verdict, event-specific feedback, strengths,
    improvements, opening rewrite, closing rewrite
  - Session recorded

Weakness Detector (speaking)
  - Same pattern as debate
  - Speaking-specific drills (e.g. "replace every 'I think' with a direct statement")

Progression System (speaking)
  - Same XP and advance/stay/drop system
  - Now knows full curriculum → references specific lessons in feedback

PERSONALIZED COACH:
  - Full knowledge of entire app curriculum via CURRICULUM_MAP
  - Knows student's name, experience, focus areas, goal
  - Knows their AI-generated coaching style, weekly goal, top skills
  - Stays in character — only coaches their chosen areas
  - References specific sections, lessons, and features by name
  - Tone calibrated: beginner=encouraging, intermediate=challenging, advanced=demanding
  - Example starter prompt responses that reference real features

2. Database (Supabase) — required for paid version
   No persistent server-side storage yet

3. Stripe Payments — not started
   Pricing: Free (3 sessions/month) / Pro $9.99/mo / School $29.99/mo

4. Usage Limits — not started
   No paywall or session counting system

5. Model UN Simulator — planned
   Full committee simulation with AI playing multiple delegates

6. Articulation Drill — planned
   Timed speaking drill for Level 1 students
